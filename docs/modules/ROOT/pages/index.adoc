== Installation

If you want to use this extension, you need to add the `io.quarkiverse.sqlpad:quarkus-devservices-sqlpad` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.sqlpad</groupId>
    <artifactId>quarkus-devservices-sqlpad</artifactId>
</dependency>
----
